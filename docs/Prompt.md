# Prompt.md

## 서비스 한 줄 설명
사용자가 챗봇과 대화하며 자신의 삶을 기록하고,
그 대화 내용을 바탕으로 장별 자서전 초안을 생성하는 안드로이드 앱

## 문제 정의
많은 사용자는 자신의 삶을 기록하고 싶어도
긴 글을 직접 쓰는 데 부담을 느낀다.
특히 무엇부터 써야 할지 모르거나,
기억을 체계적으로 정리하기 어려워한다.

## 해결 방식
앱은 인터뷰형 챗봇을 제공한다.
챗봇이 주제별 질문을 하고 사용자가 답하면,
시스템은 답변을 저장하고 핵심 기억을 구조화한 뒤,
장별 자서전 초안으로 재구성한다.

## 핵심 사용자
- 자신의 삶을 기록하고 싶은 일반 사용자
- 글쓰기에 익숙하지 않은 사용자
- 인터뷰 방식으로 쉽게 회고하고 싶은 사용자

## MVP 기능
1. 이메일 회원가입 / 로그인
2. 인터뷰 세션 생성
3. 챗봇 질문 표시
4. 사용자 답변 저장
5. 답변 기반 memory item 추출
6. memory item 기반 장별 초안 생성
7. 생성된 초안 수정 / 재생성
8. 전체 자서전 버전 저장

## MVP에서 제외
- 음성 대화
- 사진 자동 삽입
- 가족 공유
- PDF 다운로드
- 감정 점수화
- 벡터 검색 고도화
- 실시간 스트리밍 응답
- 푸시 알림

5. Book Preview Screen

- user
- session
- message
- memory_item

## 핵심 흐름
1. 사용자가 로그인한다.
2. 새 인터뷰 세션을 시작한다.
3. 챗봇 질문에 답한다.
4. 답변이 DB에 저장된다.
5. 서버가 답변에서 memory item 을 추출한다.
6. memory item 이 일정 수 이상 쌓이면 장별 초안을 생성한다.
7. 사용자는 초안을 수정하거나 다시 생성할 수 있다.
8. 최종 자서전 버전을 저장한다.

## memory item 예시
사용자 답변:
"초등학교 때 여름마다 할머니 댁에 갔는데 그때가 가장 좋았어요."

추출 결과 예시:
- period: 초등학교 시절
- place: 할머니 댁
- person: 할머니
- emotion: 행복, 그리움
- event: 여름방학 방문

## chapter 예시
- 1장 어린 시절
- 2장 가족
- 3장 학교와 성장
- 4장 힘들었던 순간
- 5장 지금의 나

## 기술 스택
- Android Kotlin
- FastAPI
- Supabase
- OpenAI API

## 보안 원칙
- OpenAI API key 는 앱에 넣지 않는다.
- OpenAI 호출은 서버에서만 수행한다.
- 사용자 데이터는 인증된 사용자 기준으로 접근 제어한다.
